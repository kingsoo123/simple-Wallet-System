Follow these instructions to get a copy of the project running on your local machine.

Prerequisites

Java Development Kit (JDK) 17 or higher

Maven 3.x

## **Installation**

Clone the repository:

git clone https://github.com/kingsoo123/simple-Wallet-System

## **Build the project:**

mvn clean install


## **Running the Application**
To start the Spring Boot application, run the following command in the project directory:

mvn spring-boot:run


After running the command, the application should start, and you can access database with this url:

http://localhost:9090/h2-console/login.jsp?jsessionid=87b382c65cbbdf868471c786e41394b6

Please note you have to change the application propertiess datasource to match your local system path
